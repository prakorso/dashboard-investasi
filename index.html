⚙️ Setup Google Apps Script
1
Buka Google Sheets → Apps Script
Extensions → Apps Script → hapus semua kode → paste isi file Code.gs
2
Deploy sebagai Web App
Deploy → New deployment → ⚙️ → Web app → Execute as: Me → Who has access: Anyone → Deploy → authorize → copy URL
3
Paste URL di Settings → Koneksi Google Sheets
Setelah tersambung semua data sync ke sheet. Login default: admin / admin123
4
Deploy ke Netlify (opsional)
netlify.com → drag & drop index.html → dapat URL publik gratis
